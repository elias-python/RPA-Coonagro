Pasta de assets para o force_cockpit.py
========================================

Salve aqui os screenshots recortados de cada elemento da UI do SAP Fiori.
Use a função `capturar_asset(nome)` do force_cockpit.py para capturar e salvar.

Exemplos de assets a criar (execute o fluxo manual e capture cada um):
  - campo_numero_nf.png     → campo de filtro por número da NF
  - btn_pesquisar.png       → botão "Pesquisar" / ícone de lupa
  - btn_acao.png            → botão da ação principal no documento
  - (adicione conforme mapeamento)

Dica: quanto mais específico o recorte (sem bordas extras), melhor a precisão.
